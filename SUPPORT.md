# Support

## Comment obtenir de l'aide

Avant de demander de l'aide, consultez les ressources disponibles :

### Documentation

- **[README principal](./README.md)** — Vue d'ensemble du projet et démarrage rapide
- **[Guide de démarrage](./docs/guide-demarrage.md)** — Installation et premiers pas
- **[Guide de contribution](./CONTRIBUTING.md)** — Comment participer au projet
- **[Documentation des scripts](./scripts/)** — Chaque dossier contient son propre README

### Chercher des réponses existantes

Avant d'ouvrir une nouvelle issue, vérifiez :

1. Les **[Issues existantes](https://github.com/lwmmedia/LM-Tools/issues)** (ouvertes et fermées)
2. Les **[Discussions](https://github.com/lwmmedia/LM-Tools/discussions)** si elles sont activées
3. La documentation du script concerné

### Ouvrir une Issue

Si vous ne trouvez pas de réponse :

- **Bug ?** Utilisez le modèle [Bug Report](./.github/ISSUE_TEMPLATE/bug_report.yml)
- **Idée ou amélioration ?** Utilisez le modèle [Feature Request](./.github/ISSUE_TEMPLATE/feature_request.yml)
- **Question générale ?** Ouvrez une issue avec le label `question`

## Ce qui n'est pas supporté

- Les modifications personnalisées ou les forks du projet
- Les problèmes liés à votre environnement système non reproductibles
- Le support des anciennes versions de Windows (antérieures à Windows 10)
- L'assistance à l'installation de Python, PowerShell ou Bash eux-mêmes

## Délais de réponse

Ce projet est maintenu sur le temps libre de ses contributeurs. Les délais de réponse peuvent varier. Merci de votre patience !

| Type de demande | Délai estimé |
|-----------------|--------------|
| Bug critique    | 48–72 heures |
| Bug mineur      | 1–2 semaines |
| Fonctionnalité  | Variable     |
| Question        | 3–5 jours    |
