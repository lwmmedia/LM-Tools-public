# Scripts Bash

Ce dossier contient des scripts Bash pour les systèmes Linux/Unix.

## Prérequis

- Système Linux/Unix ou WSL (Windows Subsystem for Linux)
- Bash shell

## Utilisation

### Rendre un script exécutable
```bash
chmod +x nom_du_script.sh
```

### Exécuter un script
```bash
./nom_du_script.sh
```

ou
```bash
bash nom_du_script.sh
```
