# Exemples d'utilisation

Ce dossier contient des exemples d'utilisation des scripts et des fichiers de test.

## Fichiers d'exemple

### exemple_texte.txt
Un fichier texte simple pour tester le script `example_analyse_texte.py`

## Utilisation

Ces fichiers sont fournis pour vous permettre de tester les scripts sans avoir à créer vos propres fichiers de test.

### Exemple : Analyser le fichier texte

```bash
cd scripts/python
python example_analyse_texte.py ../../examples/exemple_texte.txt
```

### Exemple : Créer une sauvegarde

```bash
cd scripts/bash
chmod +x example_sauvegarde.sh
./example_sauvegarde.sh ../../examples
```
