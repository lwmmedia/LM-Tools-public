## Vue d'ensemble

LM-Tools est un dépôt communautaire regroupant des outils et scripts utiles pour diverses tâches informatiques. Les scripts sont organisés par plateforme et langage pour faciliter leur utilisation.

## Catégories d'outils

### Scripts Windows
- Scripts Batch (.bat)
- Scripts PowerShell (.ps1)
- Utilitaires système Windows

### Scripts Python
- Scripts multi-plateformes
- Outils d'analyse de données
- Automatisation de tâches

### Scripts Bash
- Scripts Linux/Unix
- Outils d'administration système
- Scripts de sauvegarde et maintenance
